ASP.NET Core + MCP

EntangoApi

Backend API built for Claude Desktop integration via Model Context Protocol, and also usable as a standalone utility API for construction-focused workflows.

.NET 9 Entity Framework Core PostgreSQL Swagger/OpenAPI Claude MCP Tools

Capabilities

  • AI chat endpoints for OpenAI and Gemini
  • Document extraction from PDFs with Gemini
  • Price List CRUD and search
  • Document Store CRUD and binary download
  • Mail and SMS sending endpoints
  • Weather forecast via Open-Meteo integration

MCP Transports

stdio (local) Client starts process directly (best for local desktop usage).
HTTP/HTTPS (remote) Clients connect to /mcp URL (best for hosted/deployed usage).

Run the API

Start the main API project:

dotnet run --project EntangoApi/EntangoApi.csproj

Default local URLs:

HTTP http://localhost:5162 HTTPS https://localhost:7264 Swagger http://localhost:5162/swagger

Run the MCP Server

HTTP/HTTPS mode (default):

dotnet run --project McpServer/PriceListMcpServer.csproj

stdio mode:

dotnet run --project McpServer/PriceListMcpServer.csproj -- --stdio

Configuration Highlights

  • Database connection: ConnectionStrings:DefaultConnection
  • Provider keys: OpenAI:ApiKey, Gemini:ApiKey
  • Messaging settings: Smtp:*, Sms:*
  • MCP endpoint settings in McpServer/appsettings.json

Tech Stack

  • ASP.NET Core Web API
  • Entity Framework Core
  • Npgsql PostgreSQL provider
  • ModelContextProtocol.AspNetCore
